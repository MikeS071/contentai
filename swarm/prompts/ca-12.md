# ca-12: `contentai social` — Social Copy Generation

## Context
Depends on ca-08 (draft) and ca-11 (publish). You have: LLM client, templates, content store, publisher.

## What to Build

`contentai social <slug>`

### Process
1. Load article from `content/<slug>/article.md`
2. Load `social-copy` template
3. Send to LLM with voice.md + article → generate platform-specific copy:
   - **X/Twitter**: ≤280 chars, hook + insight + link
   - **LinkedIn**: 1-3 paragraphs, professional tone, hashtags
4. Save to `content/<slug>/social.json`:
```json
{
  "x": {"text": "...", "generated_at": "..."},
  "linkedin": {"text": "...", "generated_at": "..."}
}
```
5. Show preview, user can edit inline or regenerate

### Files
- `internal/social/generator.go` — social copy gen
- `internal/social/generator_test.go` — tests FIRST
- `cmd/social.go` — cobra command
